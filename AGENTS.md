## AGENTS

Proyecto: Generador de contraseñas (Utilidad para publicar en internet)

Rol del agente: Desarrollador web experto con 12 años de experiencia.  

Objetivo: Crear una aplicación web que permita generar contraseñas seguras y aleatorias.  

Funcionalidades de la aplicación:

- Seleccionar la longitud.
- Activar / Desactivar:
    - Mayúsculas
    - Minúsculas
    - Números
    - Caracteres especiales
- Generar contraseña aleatoria
- Copiar contraseña al portapapeles
- Indicador de fortaleza de la contraseña
- Mostrar / ocultar contraseña 


Stack de tecnologías:
- HTML5
- CSS3 (sin frameworks)
- JavaScript (vanilla, sin frameworks)

Preferencias de diseño:
- Moderno
- Responsive
- Minimalista
- Claro

Preferencias de estilos:
- No utilizr TAILWIND, NO, NO, NO!!!
- Colores azules y frios
- Uso de medidas con rem, usando un font-size base de 10px
- Uso de HTML5 y CSS3 nativo (sin frameworks) con metodologia BEM para las clases
- Usa buenas prácticas de maquetación CSS y si es necesario usa css flexbox y css grid layout
- Que la app sea responsive
- Que la app esté en idioma Español

Preferencias de código:
- No añadas dependencias externas
- El html debe ser semantico
- No uses alert, confirm, prompt, todo el feedback debe ser visual en el DOM
- No uses innerHTML, todo el contenido debe ser insertado con appendChild o previamente creado con   
  document.createElement()
- Cuidado con olvidar prevenir el default de los eventos en submit o clicks
- Prioriza el codigo legible y mantenible
- Prioriza que el codigo sea sencillo de entender
- Si el agente duda, que revise las especificaciones del proyecto y si no
  que pregunte al usuario

Estructura de archivos:
- carpeta assets
    - carpeta css
    - carpeta js
    - carpeta img
- index.html
- AGENTS.md
